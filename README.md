# Ethiopian Orthodox AI Bot 🇪🇹☦️

An AI-powered chatbot designed to provide insights, answer questions, and share information regarding the history, traditions, theology, and teachings of the Ethiopian Orthodox Tewahedo Church.

## 🚀 Features
* **Theological Q&A:** Get answers rooted in orthodox teachings and historical contexts.
* **Streamlit UI:** Clean, easy-to-use web interface for seamless interaction.
* **Hugging Face Integration:** Powered by `langchain-huggingface` for intelligent language processing.
